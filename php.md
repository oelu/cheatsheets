
## Control Structure

### Conditions
* > Greater than
* < Less than
* <= Less than or equal to
* >= Greater than or equal to
* == Equal to
* != Not equal to

### If / Else
```php
<?php
        if($items > 5) {
          echo "You get a 10% discount!";
        }
        else {
        echo "You get a 5% discount!";    
        }
?>
```
