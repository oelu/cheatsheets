# TMUX and TMUXP Cheatsheet
Cheatsheet to match my tmux configuration. 

## Settings
C-a is command key.

## Sessions
Create a new session:
    tmux new-session -s code
Attach to a session:
    tmux attach -t code
Detach from session:
    C-a d
Next session:
    C-a )
Previous session:
    C-A (
